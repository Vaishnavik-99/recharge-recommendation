# Recharge Plan Recommendation System

This project recommends the best recharge plans for users based on their usage patterns using a content-based recommendation system.

## Features
- Content-based recommendation
- Feature scaling
- User-customized recommendations

## Tech Stack
- Python, Pandas, NumPy
- Scikit-learn
- Jupyter Notebook

How to Run
1. Install requirements: pip install -r requirements.txt
2. Run notebook: recommendation.ipynb
3. Use recommend.py for custom inputs
